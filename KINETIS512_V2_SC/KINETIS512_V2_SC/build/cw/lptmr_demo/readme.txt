This project demonstrates how to use the Low Power Timer (LPTMR). It has examples of using time counting mode, pulse counting mode, and setting up multiple clock sources. The examples run one after another.
For information on how to open and use the project refer to the build/cw/cw_readme.txt file.

By default the OS-JTAG is used for the terminal output. The terminal should be configured for 115200 8-N-1.

Supported platforms:
- TWR_K60D100M

There is a project file included for each of the supported platforms. Select the project that corresponds to your hardware.
